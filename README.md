# Тест по Основам управления проектами

Интерактивный веб-тест с автоматическим переходом при правильном ответе.

## Функции

- ✅ **Мгновенная проверка** - при выборе ответа сразу видно правильный/неправильный
- ✅ **Автоматический переход** - если ответили правильно, через 1.5 сек переход к следующему вопросу
- ✅ **Прогресс-бар** - показывает сколько вопросов пройдено (например "5/42")  
- ✅ **Перемешивание** - при каждой новой попытке вопросы перемешиваются
- ✅ **42 вопроса** по основам управления проектами

## Как запустить

### Вариант 1: Просто открыть файл
Дважды кликните на `index.html` - сайт откроется в браузере.

### Вариант 2: Через локальный сервер
```bash
# Если установлен Python
python -m http.server 8000

# Затем откройте в браузере
http://localhost:8000
```

## Управление

- **Далее** - переход к следующему вопросу
- **Назад** - возврат к предыдущему
- **Новая попытка** - перемешивает вопросы и начинает заново
- **Импорт .htm** - можно загрузить свой файл с вопросами в формате HTML

## Редактирование вопросов

Вопросы находятся в файле `assets/script.js` в массиве `PM_QUESTIONS`.
