# Тесты — локальная страница

Описание: простая SPA-страница с вопросами и мгновенной подсветкой правильного ответа.

Как запустить:

1. Откройте `index.html` в браузере (двойной клик) — сайт работает как статическая страница.
2. Для локального сервера (рекомендуется для некоторых браузеров):

```bash
python -m http.server 8000
# затем откройте http://localhost:8000 в браузере
```

Где редактировать вопросы: файл `assets/script.js` — переменная `QUESTIONS`.
